В Фигме находится макет, так он должен выглядеть на самых распространённых разрешениях экранов.

### Технологии

Проект создан средствами HTML, Flex, Grid и представляет собой полноценную тренировку продающего сайта-визитки.
Необходимо [оптимизировать картинки](https://tinypng.com/), чтобы сайт загружался быстрее.
